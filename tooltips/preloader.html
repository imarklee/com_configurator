Customizing which pages the preloader is enabled on:

You may not want the preloader to be displayed on all pages, so this brief tutorial will show you how to customize which pages it is enabled on.

Step 1: Disable loading on all pages.

The first thing you need to do is set the "Enable on all pages" option to "no". Once that is done, you can proceed to the built in code editor and select the "custom footer script" option from the drop down list.
