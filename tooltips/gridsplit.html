Grid Split Explained:

Grid split refers to the ratio in which modules assigned to a particular module position. Morph will automatically float your various modules in a default fashion, unless otherwise specified. Its important to note that custom grid split options only apply when there are two modules assigned to a position. We do however plan on incorporating the Grid 960 css library to allow for even more layout options.

Grid Split Defaults:

This is a break down of the grid split defaults. There is nothing that you need to do in order to get this working. See the custom options examples on how you can customize your module grids further.

Grid Split ratio with one modules assigned
100%
Grid Split ratio with two modules assigned
50% 50%
Grid Split ratio with three modules assigned
33% 33% 33%
